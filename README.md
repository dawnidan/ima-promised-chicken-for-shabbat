# אמא, הבטחת לנו עוף לשבת 🍗

מחברת מתכונים חכמה שמונעת משברי רעב, שאלות מיותרות, וקריאות מצוקה לחסדי נעמי.

MVP ראשוני לאפליקציית React + Vite בעברית וב־RTL, עם חוויית צ'אט מרכזית לתכנון ארוחות שבת, סידור מתכונים ורשימת קניות מוכנה להעתקה לוואטסאפ.

## עקרונות המוצר

- דף בית יחיד
- mobile-first
- ממשק עברי ב־RTL
- fallback מקומי לצד שכבת backend ראשונה
- בלי חיפוש, פילטרים, התחברות או מסד נתונים בשלב הזה
- כל האתר מעוצב לפי "סגנון מתכון 1": מחברת מתכונים חמימה, מאוירת, ביתית ולא טכנולוגית מדי

## מה יש בגרסה הזו

- כותרת מאוירת וחמימה
- חלון צ'אט מרכזי
- שדה הודעה וכפתור שליחה
- שכבת backend ראשונה לצ'אט אמיתי
- fallback מקומי אם אין עדיין מפתח API
- תפריט ורשימת קניות שמופיעים בסיום השיחה

## הרצה מקומית

1. התקינו Node.js.
2. העתיקו את `.env.example` לקובץ `.env`.
3. מלאו את `OPENAI_API_KEY` אם רוצים להפעיל AI אמיתי.
4. הריצו בשני חלונות טרמינל:

```bash
npm install
npm run dev
```

```bash
npm run dev:api
```

5. פתחו את הכתובת ש־Vite יציג, בדרך כלל `http://localhost:5173`.

## חיבור מאגר המתכונים שלך

השלב הבא הוא להכניס את קובצי המתכונים האמיתיים שלך אל `recipe-corpus/`, להעלות אותם ל־vector store, ולהגדיר את `OPENAI_VECTOR_STORE_ID` כדי שהמודל ישלוף תשובות מהמאגר שלך.

אחרי שיש קבצים בתיקייה:

```bash
npm run upload:recipes
```

הפקודה תחזיר מזהה `OPENAI_VECTOR_STORE_ID` שאותו צריך להעתיק לקובץ `.env`.
