# אמא, הבטחת לנו עוף לשבת 🍗

MVP ראשוני לאפליקציית React + Vite בעברית וב־RTL, עם חוויית צ'אט מרכזית לתכנון ארוחות שבת, סידור מתכונים ורשימת קניות מוכנה להעתקה לוואטסאפ.

## עקרונות המוצר

- דף בית יחיד
- mobile-first
- ממשק עברי ב־RTL
- נתוני mock מקומיים בלבד
- בלי חיפוש, פילטרים, התחברות או מסד נתונים בשלב הזה
- כל האתר מעוצב לפי "סגנון מתכון 1": מחברת מתכונים חמימה, מאוירת, ביתית ולא טכנולוגית מדי

## הרצה מקומית

1. התקינו Node.js במחשב אם הוא עדיין לא מותקן.
2. מתוך תיקיית הפרויקט הריצו:

```bash
npm install
npm run dev
```

3. פתחו את הכתובת ש־Vite יציג בטרמינל, בדרך כלל `http://localhost:5173`.

## מבנה הפרויקט

```text
src/
  components/
  data/
  utils/
  styles/
docs/
```

## מה יש בגרסה הזו

- כותרת מאוירת וחמימה
- שלושה כפתורי התחלה מהירה עם תגובות mock
- חלון צ'אט מרכזי
- שדה הודעה וכפתור שליחה
- רשימת קניות מוכנה להעתקה לוואטסאפ
