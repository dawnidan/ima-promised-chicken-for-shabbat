# Prompt Examples

## Menu planning

- עזור לי לבנות תפריט לשבת
- יש לי 6 אורחים, מה כדאי להכין?
- אני צריכה תפריט צמחוני לשישי בערב

- הנה מתכון מבולגן, תסדרי לי אותו
- הפכי את זה לרשימת רכיבים ושלבים

## Shopping list

- תאחדי לי כמויות כפולות מאותו מוצר
