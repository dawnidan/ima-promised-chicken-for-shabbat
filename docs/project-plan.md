# Project Plan

## MVP goal

Create a warm, mobile-first homepage that helps a user start a Shabbat-planning conversation immediately.

## Phase 1

- Build the homepage shell
- Add illustrated header
- Add quick prompt buttons
- Add mock chat flow
- Add WhatsApp-ready shopping list copy action

## Later phases

- Real AI responses
- Recipe ingestion and cleanup
- WhatsApp archive import
- Saved recipe collection
- Generated shopping lists from selected recipes
- Family preference memory
